# Commands

## Command to create app with vite
- npm create vite@latest

## To install npm
- npm install

## To install react router dom
- npm install react-router-dom@6

## To install styled components
- npm i styled-components

## To install react icons
- npm i react-icons

## To run vite
- npm run dev
